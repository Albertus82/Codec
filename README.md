Codec
=====
```ruby
Usage: codec <mode> <algorithm> "Text to encode/decode"

Mode:
    e    Encode
    d    Decode

Algorithms:
    Base16
    Base32
    Base64
    MD2
    MD5
    SHA-1
    SHA-256
    SHA-384
    SHA-512

Example: codec e base64 "Lorem ipsum dolor sit amet"
```
